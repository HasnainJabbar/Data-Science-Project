**📧 # Spam Email Classifier using Machine Learning**

🚀 Project Overview

This project is a Spam Email Classification System built using Python and Machine Learning techniques. The objective of this project is to classify emails as Spam or Not Spam by training and evaluating multiple machine learning models.

The project focuses on comparing different algorithms to determine which performs best for text classification tasks.

---

✨ Features

- Email text preprocessing and cleaning
- Feature extraction and vectorization
- Training multiple machine learning models
- Model comparison and evaluation
- Data visualization for performance analysis
- Confusion Matrix generation for classification analysis

---

🤖 Machine Learning Models Used

The following machine learning algorithms were trained and evaluated:

- Naive Bayes
- Logistic Regression
- Decision Tree
- Support Vector Machine (SVM)
- Random Forest

---

📊 Visualizations Included

Bar Chart Comparison

Used to compare model performance metrics such as accuracy and evaluation scores.

Pie Charts

Used for visualizing:

- Dataset distribution
- Spam vs Non-Spam predictions

Confusion Matrix

Used to analyze:

- Correct classifications
- False Positives
- False Negatives
- Overall model behavior
